# Gestion d'une Bibliothèque avec Procédures PL/SQL

## Description
Projet réalisé dans le cadre du cours de **Base de Données 2** — Licence
Statistique et Informatique (LSI), Semestre 4  
Université Nazi Boni, Bobo-Dioulasso — Année académique 2025-2026  
Enseignant : **Dr Go Issa TRAORE**

Ce projet implémente un système de gestion de bibliothèque universitaire
en PL/SQL sous Oracle, avec des procédures stockées et un menu interactif.

## Objectifs
1. Concevoir et manipuler des procédures stockées en PL/SQL
2. Implémenter un menu interactif en environnement Oracle
3. Gérer les livres, abonnés et emprunts
4. Mettre en œuvre la gestion des erreurs
